    Class Log

    Lightweight, host-agnostic logger.

    • Works in plain .NET via Console/Debug.
    • Detects Unity at runtime via reflection and forwards to UnityEngine.Debug.
    • Exposes a pluggable sink/event so hosts can capture logs.
