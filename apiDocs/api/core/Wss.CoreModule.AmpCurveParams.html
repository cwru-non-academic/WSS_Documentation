    Class AmpCurveParams

    Piecewise amplitude curve parameters used to map mA to 0..255. mA <= LowThreshold: out = pow(mA / LowConst, ExpPower) + 1 mA > LowThreshold: out = ((mA + LinearOffset) / LinearSlope) + 1
